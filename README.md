# Projeto Ciência de Dados - FASE 2

## Como abrir o projeto no Orange Data Mining

1. Acesse o repositório pelo link:  
   https://github.com/MariaClaret18/projeto-ciencia-de-dados-IA-FASE2

2. Clique no botão **Code** e selecione **Download ZIP**.

3. Descompacte o arquivo baixado.

4. Dentro da pasta descompactada, você encontrará:
   - o arquivo do projeto do Orange (**.ows**)  
   - o arquivo **CSV** utilizado na análise

Abra o arquivo `.ows` no Orange para visualizar o workflow completo.  
Se o CSV não carregar automaticamente, selecione-o manualmente no widget **File**.
